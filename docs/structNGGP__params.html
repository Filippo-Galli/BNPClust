Detailed Description

Struct for storing parameters of the NGGP (Normalized Generalized Gamma Process).

This struct holds the parameters for the NGGP model, including the total mass parameter a, the second parameter sigma, and the third parameter tau.

The parameters are stored as public members for easy access.
